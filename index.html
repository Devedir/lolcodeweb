What is LOLCODEWEB?

As this Wikipedia article says:

LOLCODE is an esoteric programming language inspired by lolspeak, the language expressed in examples of the lolcat Internet meme.
The language was created in 2007 by Adam Lindsay, a researcher at the Computing Department of Lancaster University.

This website provides a dialect of this language that can run in your browser.
it does so by compiling the code to JavaScript, which is then run unless some errors are caught at compile time.
You can see your object code (or happy little errors) in the browser console.

This project was created as a college assignment at AGH UST.
